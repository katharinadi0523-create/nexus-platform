# Nexus AI Platform (中国电子云 | AI平台)
智能体 · 赋能企业的每一步

## 📖 项目简介 (Introduction)

Nexus AI Platform 是中国电子云 AI 平台的核心组成部分——应用开发平台的高保真前端原型。

本项目旨在展示下一代企业级 AI 原生应用开发体验。通过一站式的工具链，帮助企业用户快速构建、编排、管理和发布 AI 智能体（Agents）。平台集成了应用广场、可视化工作流编排、多模态知识库管理以及模型全生命周期管理等核心能力，为企业提供从模型到应用落地的全链路解决方案。

## ✨ 核心功能 (Key Features)

### 🤖 应用广场 (App Marketplace)
汇集丰富的高质量智能体模板，支持一键复用与二次开发。采用清爽的卡片式布局与横向轮播设计，提供极致的视觉体验。

### 🎨 智能体编排 (Agent Workflow)
基于 React Flow 构建的可视化工作流编辑器。支持大模型、知识检索、意图识别、逻辑分支等全量节点库，提供所见即所得的拖拽编排体验。

### 📚 知识库 (Knowledge Base)
企业级 RAG 解决方案。除支持 PDF/Word 等文档外，特色支持表格型数据的自动解析与表头识别，专为结构化数据问答优化。

### 🧠 模型开发与服务 (Model Dev)
提供大模型全生命周期管理界面的交互演示，直观展示模型服务的运行状态、来源与调用详情。

### 🔧 开发者套件
包含 API Key 细粒度权限管理与可视化的 Trace 链路追踪（UI Mock）。

## 🛠 技术栈 (Tech Stack)

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Shadcn UI
- **Visual Flow**: React Flow
- **Icons**: Lucide React

## 🚀 快速开始 (Getting Started)

```bash
# 1. 克隆项目
git clone https://github.com/katharinadi0523-create/nexus-platform.git

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev
```

打开浏览器访问 [http://localhost:3000](http://localhost:3000) 即可查看效果。
